- .m 文件为 matlab 文件
- 不加分号自动输出结果，加分号不输出结果
- % 为注释, ctrl+r 一键注释，ctrl+t 取消注释
- clear 清除工作区所有变量
- clc 清空命令行
- 分号也可以分隔行
- ~= 不等于

- 输出
	- Disp ("xxx") 输出字符串
	- Disp（a）输出变量
	- disp 加不加分号都会输出
	- 向量用空格、逗号分隔行元素，用分号分割列元素
- 合并字符串
	- Strcat (str 1, str 2, str 3...)
	- [str 1, str 2, str 3...]
	- [str 1  str 2  str 3...]
- 数字转化为字符串
	- Num 2 str (数字)
- 输入
	- A = input ("xxx: ")  输入值赋值给 A，并输出 "xxx: "
- 求和
	- 行列向量
		- E = [1, 2, 3]
		- Sum（E）行列向量直接求和
	- 矩阵行列求和
		- E = [1, 2; 3, 4; 5, 6]
		- Sum (E) 矩阵默认按列求和, 得到行向量
		- Sum（E，1）指定按列求和
		- Sum（E, 2）指定按行求和
	- 矩阵整体求和
		- Sum (sum (E))
		- Sum (E (:))  E (:) 为 E 的按列拼接的列向量
- 取指定元素
	- A (2，1) 取第二行第一列的元素
	- A (2, :) 取第二行所有列的元素
	- A (:, 5) 取所有行第五列的元素
	- A ([2, 5], :) 取第二行和第五行
	- A (2 : 5,:) 取第二行到第五行，默认步长为一
	- A (2：2：5，:) 取第二行到第五行，步长为二
	- A (2: end,:) 取第二行到最后一行
	- A (2: end-1,:) 取第二行到倒数第二行
- 生成矩阵
	- Zeros（n, m) 生成 n 行 m 列的全为 0 的矩阵
	- A + rand (n, m) * (b-a)生成 n 行 m 列的矩阵（每一个元素范围为 n~m）
	- Randi ([a, b], n, m) 生成 n 行 m 列矩阵（每一个元素范围为 a~b）
	- Randi ([a, b]) 在 a~b 中随机生成一个数
- 数据处理
- 结果绘图
	- Axis ([a, b, c, d])；绘制坐标图 x 取值 a 到 b，y 取值 c 到d
	- Box on; 封闭坐标图边界
	- Plot（A (i)，B (i),'r'） 描点，横坐标为 A (i), 纵坐标为 B (i)
	- Hold on; 在原来的图形上继续绘制
