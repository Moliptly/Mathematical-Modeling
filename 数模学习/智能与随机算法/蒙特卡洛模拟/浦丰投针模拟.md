## 用途
- 用随机数模拟实验
- 用计算机编程模拟实验过程

## 预备知识
- Rand (5,4)生成五行四列的矩阵（每一个元素都是 0~1 之间的随机数）
- Rand (3)生成三行三列的方阵
- a + rand (n, m) * (b-a)生成 n 行 m 列的矩阵（每一个元素范围为 n~m）
- Unifrnd (a, b, n, m) 等价于上面的式子

## 代码部分
```c++
//matlab 模拟浦丰投针
l = 0.520; //针的长度
a = 1.314; //平行线间距
n = 1000000; //试验次数
m = 0; //针与平行线相交的次数
x = rand(1, n) * a / 2; //随机生成针中点到线的距离
phi = rand(1, n) * pi; //随机生成针与线的夹角

for i = 1 : n
	if x(i) <= l / 2 * sin(phi(i))
		m = m + 1;
	end
end

p = n / m; //相交的频率


//优化
//将n变小，多次重复求平均值
l = 0.520; //针的长度
a = 1.314; //平行线间距
n = 1000000; //试验次数
result = zeros(100, 1);
for num = 1 : 100
	m = 0;
	x = rand(1, n) * a / 2;
	phi = rand(1, n) * pi;
	for i = 1 : n
		if x(i) <= l / 2 * sin(phi(i))
			m = m + 1;
	p = n / m;
	mypi = (2 * l) / (p * a);
	result(num) = mypi; //存储计算结果
end
mymeanpi = mean(result); //计算mypi的平均值

```
