## 题目描述
- 你参加一档电视节目，节目组提供了 ABC 三扇门，主持人告诉你，其中一扇门后边有辆汽车，其它两扇门后是空的。假如你选择了 B 门，这时，主持人打开了 C 门，让你看到 C 门后什么都没有，然后问你要不要改选 A 门？

## 模拟代码
```c ++
n = 1000000; // 试验次数
a = 0; //不改变主意时赢得汽车次数
b = 0; //改变主意时赢得汽车次数
for i = 1 : n
	x = randi([1, 3]); //汽车所在门号 
	y = randi([1, 3]); //自己选择的门号
	if x == y
		a = a + 1; //不用改变主意
	else b = b + 1; //需要改变主意
	end
end

```
