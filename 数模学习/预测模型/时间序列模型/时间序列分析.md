# 时间序列分解
## 概念
- 同一对象在不同时间连续观察所取得的数据
- 两要素：==时间==要素、==数值==要素
- 两分类：==时期==序列（一个数据对应一段时期）、==时点==序列（一个数据对应一个时刻）
- 时期序列可以相加：表示更长的量的积累
- 时点序列不可相加：无意义
## 用法
- 描述过去、分析规律、预测未来
## 步骤 (主要用spss)
- 做时间序列图
- 判断时间序列包含的变动的成分
- 时间序列分解
- 建立时间序列分析模型
- 预测未来的指标数值

## 重点
- 时间序列分解:（有周期性/周期小于年才能用）
- 长期趋势变动 T（持续上升/下降）
- 季节变动 S（以季节/月份/周为周期）  工具：百度指数
- 循环变动 C（以若干年为周期）
- 不规则变动 I（不可预知无规律）

- ==叠加==模型：Y = T + S + C + I (四种变动==相互独立==/波动==不随== 时变)
- ==乘积==模型：Y = T x S x C x I (四种变动==相互影响==/波动==随==时变)

- 主要使用 spss 进行时间序列分解
- 缺失值在头尾，直接删除
- 缺失值在中间，替换缺失值（5 种方法）

## 实操
- 建立时间序列模型
- 主要用 spss，提供两类模型（指数平滑模型/ARIMA 模型），自动检测异常值
- 注意点：
	- 缺失值补全：转换-替换缺失值
	- 定义时间变量：数据-定义日期和时间-选择个案-设置起始年份
	- 绘制时间序列图：分析-时间序列预测-序列图-选择变量（因变量）-选择时间轴标签（定义的时间变量）
	- 季节性分解：分析-时间序列检测-季节性分解-选择变量（因变量）-选择模型类型-选择移动平均值权重（周期是奇数选所有点相等、否则选二）
	- ERR：残差 I、SAS：T+C+I、SAF: S、STC：T+C
- 结果解读
	- 加法季节因子的和为 0，乘法季节因子的乘积为 1
	- 加法季节因子 ![[1705923492626.png]]
		- 解读：1，2 季度季节因子为正，3，4 季度季节因子为负，说明 1，2 季度平均销量高于 3，4 季度；2 季度平局销量高于全年平均水平 20.93 件，4 季度平均销量低于全年平均水平 19.727 件
	- 乘法季节因子 ![[1705923731456.png]]
		- 解读：2 季度的季节因子为 1.206，则 2 季度的平均销量是全年平均销量的 1.206 倍
- 画出分解后的时序图：分析-时间序列预测-序列图-选择变量（==多个==因变量）-选择时间轴标签（定义的时间变量）
