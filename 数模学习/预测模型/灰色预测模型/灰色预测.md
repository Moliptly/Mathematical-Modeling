## 概念
- 灰色系统
	- 白色系统：系统的信息完全明确
	- 灰色系统：系统信息部分明确，部分未知
	- 黑色系统：系统信息都未知
- 在一定范围内变化的与时间有关的灰色过程进行预测
- 数据满足==非负性==

## 适用条件
- 数据是以年份度量的非负数据（月份或者季度数据一定要用时间序列模型）
- 数据通过准指数检验 (除了前两期，后面至少 85-90%的期数的光滑比要低于 0.5)
- 数据期数较短，一般小于 10 期（期数过长用时间序列模型）
- 数据之间的关联性不强

### GM (1,1)模型
- 形式：$x^{(0)}(k)+az^{(1)}(k)=b$ 其中 $b\text{ 表示灰作用量,}-a\text{ 表示发展系数。}$
- 含义：第一个参数表示1 阶，第二个参数表示1 个变量
- 步骤
	- 通过累加数据来削弱随机性
	- 建立累加数据的方程模型
	- 根据模型预测累加数据
	- 将累加数据累减得到原始数据
- 原理
	- 原始数据序列：$x^{(0)}=\begin{pmatrix}x^{(0)}(1),x^{(0)}(2),\cdots,x^{(0)}(n)\end{pmatrix}$
	- 累加数据序列：$x^{(1)}=\left(x^{(1)}(1),x^{(1)}(2),\cdotp\cdotp,x^{(1)}(n)\right)$ 其中 $x^{(1)}(m)=\sum_{i=1}^{m}x^{(0)}(i),m=1,2,\cdotp\cdotp n$
	- 紧邻均值生成数列：$z^{(1)}=\left(z^{(1)}(2),z^{(1)}(3),\cdots,z^{(1)}(n)\right)$ 其中 $z^{(1)}(m)=\delta x^{(1)}(m)+(1-\delta)x^{(1)}(m-1)\mathrm{~},m=2,3,\mathrm{~}\cdotp\cdotp\cdotp n\text{且}\delta=0.5$
	- $x^{(0)}(k)+az^{(1)}(k)=b$ 为离散型的方程，可以用连续性微分方程近似：${ \frac { d x ^ { ( 1 ) }(t)}{dt}}=-\hat{\boldsymbol{a}}\boldsymbol{x}^{(1)}(t)+\hat{\boldsymbol{b}}$ 被称为 GM (1,1)模型的白化方程，用白化方程预测
	- 求解微分方程得 $\overset{\wedge}{\operatorname*{x}}^{(1)}(t)=\left[x^{(0)}(1)-\frac{\widehat{b}}{\widehat{a}}\right]e^{-\hat{a}(t-1)}+\frac{\widehat{b}}{\widehat{a}}$ 进而得 $\overset{\wedge}{x}^{(0)}(m+1)=\overset{\wedge}{x}^{(1)}(m+1)-\overset{\wedge}{x}^{(1)}(m)=(1-e^{\hat{a}})\left[x^{(0)}(1)-\frac{\hat{b}}{\hat{a}}\right]e^{-\hat{a}m},m=1,2,\cdots,n-1$
	- 用上式取 $m>=n$ 即可进行预测
- 参数求解
	- 法一：$x^{(0)}(k)+az^{(1)}(k)=b$ 表示为 $Y=Bu$ 则 $\hat{\boldsymbol{u}}=\begin{pmatrix}\hat{a}\\\hat{b}\end{pmatrix}=(\boldsymbol{B}^T\boldsymbol{B})^{-1}\boldsymbol{B}^T\boldsymbol{Y}$ 其中 $\boldsymbol{u}=(a,b)^T,\boldsymbol{Y}=\begin{bmatrix}x^{(0)}(2)\\x^{(0)}(3)\\\vdots\\x^{(0)}(n)\end{bmatrix},\boldsymbol{B}=\begin{bmatrix}-z^{(1)}(2)&1\\-z^{(1)}(3)&1\\\vdots&\vdots\\-z^{(1)}(n)&1\end{bmatrix}$ 求出 u 即可==求出 a，b== 的估计值
	- 法二：$x^{(0)}(k)~=~-az^{(1)}(k)+b~\Rightarrow~y~=~kx~+~b$ 用回归求出 k，b 即可求出 ==a，b 的估计值==
- 准指数规律检验
	- 只有具有准指数规律的数据才能使用 GM(1,1)模型
	- $\dot{\text{累加}r\text{ 次的序列为:}} x ^ { ( r ) }=(x^{(r)}(1),x^{(r)}(2),\cdots,x^{(r)}(n))$
	- $\text{定义级比：}\sigma(k)=\frac{x^{(r)}(k)}{x^{(r)}(k-1)},k=2,3,\cdotp\cdotp,n.$
	- $\forall k,\sigma(k)\in[a,b],\text{ 且区间长度}\delta=b-a<0.5\text{,则称累加}r\text{次后的序列具有准指数规律。}$
	- $\text{序列}x^{(1)}\text{的级比：}\sigma(k)=\frac{x^{(1)}(k)}{x^{(1)}(k-1)}=\frac{x^{(0)}(k)+x^{(1)}(k-1)}{x^{(1)}(k-1)}=\frac{x^{(0)}(k)}{x^{(1)}(k-1)}+1$
	- 序列 $x^{(1)}$ 的光滑比：$\rho\left(k\right)=\frac{x^{\left(0\right)}(k)}{x^{\left(1\right)}(k-1)}$
	- $\text{即}\forall k,\text{ 区间长度}\delta<0.5\text{,只需要只需要保证}\rho(k)\in(0,0.5)\text{即可}$
