#### 作业分析
- 系统状态方程：$\dot{x}(t)=ax(t)+u(t),\quad x(0)=x_0$
- 性能指标函数：$J=\frac12fx^2(t_f)+\frac12\int_0^{t_f}\left[qx^2(t)+ru^2(t)\right]dt$
- Hamiltonian函数：$H(x,u,\lambda)=\frac12(qx^2+ru^2)+\lambda(ax+u)$

#### 实现过程
- 为了最小化Hamiltonian，对控制变量u求导并令其为零有：$$\frac{\partial H}{\partial u}=ru+\lambda=0\implies u=-\frac\lambda r$$
- 根据Costate方程有：$$\dot{\lambda}=-\frac{\partial H}{\partial x}=-qx-\lambda a$$
- 结合上述方程，写为矩阵形式为：$$\begin{bmatrix}\dot x\\\dot\lambda\end{bmatrix}=\begin{bmatrix}a&-\frac1r\\-q&-a\end{bmatrix}\begin{bmatrix}x\\\lambda\end{bmatrix}$$
- 求解矩阵特征值$\lambda$：$\lambda_{1,2}=\frac{-a\pm\sqrt{a^2-4\frac qr}}2$
- 求解状态和协态方程
	- 通解形式
		- $x(t)=C_1e^{\lambda_1t}+C_2e^{\lambda_2t}$
		- $\lambda(t)=K_1e^{\lambda_1t}+K_2e^{\lambda_2t}$
	- 边界条件
		- $x(0)=x_0,\quad\lambda(t_f)=fx(t_f)$
- 将边界条件代入通解，即可解出$C_1,C_2,K_1,K_2$，得最优控制为：$$u^*(t)=-\frac{\lambda(t)}r$$
- 自拟参数编程求解如下
```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# 参数设置
a = 0.5
q = 1.0
r = 1.0
f = 1.0
x0 = 1.0
tf = 10.0

# 特征值计算（使用复数）
lambda1 = (-a + np.sqrt(complex(a**2 - 4 * q / r))) / 2
lambda2 = (-a - np.sqrt(complex(a**2 - 4 * q / r))) / 2

# 常数解
A = np.array([[1, 1], [lambda1, lambda2]])
B = np.array([x0, f * (x0 * (np.exp(lambda1 * tf) - np.exp(lambda2 * tf)) / (lambda1 - lambda2))])

C = np.linalg.solve(A, B)

# 状态方程的解
def x(t):
    return np.real(C[0] * np.exp(lambda1 * t) + C[1] * np.exp(lambda2 * t))

def lambda_(t):
    return np.real(C[0] * lambda1 * np.exp(lambda1 * t) + C[1] * lambda2 * np.exp(lambda2 * t))

# 时间点
t = np.linspace(0, tf, 1000)
x_vals = x(t)
lambda_vals = lambda_(t)
u_vals = -lambda_vals / r

# 绘图
plt.figure(figsize=(12, 8))
plt.subplot(3, 1, 1)
plt.plot(t, x_vals, label='x(t)')
plt.xlabel('时间 t')
plt.ylabel('状态 x(t)')
plt.legend()

plt.subplot(3, 1, 2)
plt.plot(t, lambda_vals, label='lambda(t)', color='orange')
plt.xlabel('时间 t')
plt.ylabel('协态 λ(t)')
plt.legend()

plt.subplot(3, 1, 3)
plt.plot(t, u_vals, label='u(t)', color='green')
plt.xlabel('时间 t')
plt.ylabel('控制 u(t)')
plt.legend()

plt.tight_layout()
plt.show()

```

#### 作业结果
![[1721791502357.png]]
- 图1显示了状态 x(t) 随时间的变化
    - 初始时刻 x(0) = 1
    - 随着时间的推移，状态 x(t) 呈现出一种振荡的趋势，在整个时间段内，状态逐渐趋向于稳定
- 图2显示了协态 λ(t) 随时间的变化
    - 初始时刻 λ(0) 较高，然后随着时间的推移，协态变量经历了几个周期的振荡
    - 协态的变化趋势和状态变量 x(t) 类似，但振荡的幅度和频率不同，这说明协态和状态之间存在一定的相互影响，但不完全一致
- 图3显示了控制 u(t) 随时间的变化。
    - 初始时刻，控制 u(t) 较低，然后逐渐上升，达到一个峰值后再下降，呈现出一种先增后减的趋势
    - 控制变量的变化趋势与协态 λ(t) 直接相关，因为控制 u(t) 是由协态 λ(t) 除以 r 得到的
    - 控制变量的这种变化趋势有助于将状态 x(t) 引导到目标状态，尽可能减小性能指标 J
