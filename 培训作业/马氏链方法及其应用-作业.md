**题目一**
**(1)**:由题意知，转移矩阵P为$$P=\begin{bmatrix}0.40&0.25&0.13&0.22\\0.33&0.27&0.17&0.23\\0.24&0.15&0.43&0.18\\0.03&0.13&0.27&0.57\end{bmatrix}$$
初始状态向量为$a=(100,100,100,100)$
初始比例向量为$a_0=(0.25,0.25,0.25,0.25)$
稳定状态计算如下，根据定义有$$\pi p = \pi, \pi e = 1(即\pi_1+\pi_2+\pi_3+\pi_4=1)$$
列出如下方程组：$$\begin{cases}0.4\pi_1+0.33\pi_2+0.24\pi_3+0.03\pi_4=\pi_1\\0.25\pi_1+0.27\pi_2+0.15\pi_3+0.13\pi_4=\pi_2\\0.13\pi_1+0.17\pi_2+0.43\pi_3+0.27\pi_4=\pi_3\\0.22\pi_1+0.23\pi_2+0.18\pi_3+0.57\pi_4=\pi_4\\\pi_1+\pi_2+\pi_3+\pi_4=1\end{cases}$$
使用Matlab求解上述方程组
```matlab
% 定义转移矩阵
P = [0.40 0.25 0.13 0.22; 
     0.33 0.27 0.17 0.23; 
     0.24 0.15 0.43 0.18; 
     0.03 0.13 0.27 0.57];

% 定义方程组 Ax = b
A = [P' - eye(4); ones(1,4)];
b = [zeros(4,1); 1];

% 求解线性方程组
pi = A\b;
```
解得：$\pi=(0.2246,0.1886,0.2615,0.3253)$ 即为长期后稳定的比例

**求解达到稳定的天数**：
```matlab
% 定义转移矩阵  
P = [0.40 0.25 0.13 0.22;   
     0.33 0.27 0.17 0.23;   
     0.24 0.15 0.43 0.18;   
     0.03 0.13 0.27 0.57];  
  
% 定义误差阈值  
tol = 1e-6;  
  
% 初始化  
days = 0;  
P_current = P;  
P_next = P * P;  
diff = max(max(abs(P_next - P_current)));  
  
% 迭代直到转移矩阵稳定  
while diff > tol  
    P_current = P_next;  
    P_next = P_current * P;  
    diff = max(max(abs(P_next - P_current)));  
    days = days + 1;  
end  
  
% 打印结果  
fprintf('达到稳定状态所需的天数：%d\n', days);  
  
% 稳定状态向量

b = [zeros(4,1); 1];  
pi = A\b;  
  
% 打印长期稳定状态  
fprintf('长期稳定状态向量:\n');  
disp(pi);
```

**题目二**
**(1)**: 
**M/M/m 排队模型**
平均到达率$\lambda=40$ (人/小时)
平均离开率(服务率）$\mu=60/4=15$（人/小时）
流量强度$\rho=\frac\lambda{s\mu}$
为了系统稳定，有$\rho<1$,即$s>\frac\lambda\mu=40/15 \approx2.67$
所以需要至少需要3个柜台

**(2)**:柜台数量为4个，$s=4$
**计算系统中顾客的概率分布 $P_0$ 和 $P_n$​**:
计算$P_0$:$$P_0=\left[\sum_{n=0}^{s-1}\frac{(\lambda/\mu)^n}{n!}+\frac{(\lambda/\mu)^s}{s!}\frac1{1-\rho}\right]^{-1}$$
对于$s=4$:$$P_0=\left[\sum_{n=0}^3\frac{(40/15)^n}{n!}+\frac{(40/15)^4}{4!}\frac1{1-2/3}\right]^{-1}$$
**计算平均排队等待时间** $W_q$​:$$W_q=\frac{P_0\cdot(\lambda/\mu)^s\cdot\rho}{s!\cdot(1-\rho)^2\cdot\lambda}$$
**计算平均系统时间**: $W=W_q+\frac1\mu$

**计算平均排队中的顾客数**: $L_q=\lambda\cdot W_q$

**计算系统中的平均顾客数**: $L=L_q+\frac\lambda\mu$

MATLAB求解：
```matlab
% 定义参数
lambda = 40; % 平均到达率
mu = 15; % 平均服务率
s = 4; % 柜台数

% 计算流量强度
rho = lambda / (s * mu);

% 计算P0
P0_inv = 0;
for n = 0:s-1
    P0_inv = P0_inv + (lambda / mu)^n / factorial(n);
end
P0_inv = P0_inv + (lambda / mu)^s / (factorial(s) * (1 - rho));
P0 = 1 / P0_inv;

% 计算平均排队等待时间 Wq
Wq = (P0 * (lambda / mu)^s * rho) / (factorial(s) * (1 - rho)^2 * lambda);
% 计算平均系统时间 W
W = Wq + 1 / mu;
% 计算平均排队中的顾客数 Lq
Lq = lambda * Wq;
% 计算系统中的平均顾客数 L
L = Lq + lambda / mu;

% 打印结果
fprintf('平均排队等待时间 Wq: %.4f 小时\n', Wq);
fprintf('平均系统时间 W: %.4f 小时\n', W);
fprintf('平均排队中的顾客数 Lq: %.4f\n', Lq);
fprintf('系统中的平均顾客数 L: %.4f\n', L);

```
