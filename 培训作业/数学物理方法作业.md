#### 作业分析
要判断二阶偏微分方程的类型，需要将方程写成标准形式并确定其系数矩阵的判别式。一般的二阶偏微分方程可以写成如下形式：$$A u_{xx} + 2B u_{xy} + C u_{yy} + \text{(lower-order terms)} = 0$$
我们通过计算判别式 $\Delta = B^2 - AC$来判断方程的类型：

- 如果 $\Delta > 0$，则方程是双曲型的。
- 如果 $\Delta = 0$，则方程是抛物型的。
- 如果 $\Delta < 0$，则方程是椭圆型的。

**方程 (1)**
$$u_{xx} + 4u_{xy} - 3u_{yy} + 2u_x + 6u_y = 0$$

提取系数：
- A = 1
- B = 2
- C = -3
计算判别式： $\Delta = B^2 - AC = 2^2 - (1)(-3) = 4 + 3 = 7$
因为 $\Delta > 0$，所以方程 (1) 是**双曲型**的。

**方程 (2)**
$$u_{xx} + (1 + y^2) u_{yy} + xu_x + yu_y = 0$$
提取系数：

- $A = 1 + x^2$
- B = 0 （因为没有 $u_{xy}$​ 项）
- $C = 1 + y^2$
计算判别式： $\Delta = B^2 - AC = 0^2 - (1 + x^2)(1 + y^2) = -(1 + x^2)(1 + y^2)$
因为 $\Delta < 0$，所以方程 (2) 是**椭圆型**的。

**方程 (3)**
$$u_{xx} + xy u_{yy} = 0$$
提取系数：
- A = 1
- B = 0
- C = xy
计算判别式： $\Delta = B^2 - AC = 0^2 - (1)(xy) = -xy$
方程 (3) 的类型依赖于 xy 的符号：
- 如果 xy > 0，则 $\Delta < 0$，方程是椭圆型的。
- 如果 xy = 0，则 $\Delta = 0$，方程是抛物型的。
- 如果 xy < 0，则 $\Delta > 0$，方程是双曲型的。

综上所述：
1. 方程 (1) 是双曲型的。
2. 方程 (2) 是椭圆型的。
3. 方程 (3) 根据 xy 的符号可以是双曲型、抛物型或椭圆型。
