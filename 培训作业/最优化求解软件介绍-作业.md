#### 作业分析
- **问题描述**：需要决定如何从5个工厂向4个仓库分配产品，以满足仓库的需求并最小化总成本。总成本包括运输成本和工厂的固定成本。如果关闭某些工厂，可以减少固定成本，但可能增加运输成本。
- **变量定义**：
	- $x_{ij}$​: 从工厂 i 向仓库 j 运输的产品数量
	- $y_i$​: 工厂 i 是否开放 (1 表示开放，0 表示关闭)
	- $c_{ij}$​ 是工厂 i 到仓库 j 的运输成本
	- $f_i$​ 是工厂 i 的固定成本
- **目标函数**（运输成本 + 固定成本）：$$\text{Minimize}\quad\sum_{i=1}^5\sum_{j=1}^4c_{ij}x_{ij}+\sum_{i=1}^5f_iy_i$$
- **约束条件**：
	- 1.每个仓库的需求必须得到满足：$$\sum_{i=1}^5x_{ij}=d_j\quad\forall j\in\{1,2,3,4\}$$
	- 2.每个工厂的供应不能超过其生产能力：$$\sum_{j=1}^4x_{ij}\leq s_iy_i\quad\forall i\in\{1,2,3,4,5\}$$
	- $x_{ij}\geq0\quad\forall i,j \quad y_{i}\in\{0,1\}\quad\forall i$

#### 实现过程
- 根据题意有
	- 运输成本矩阵$$c=\begin{bmatrix}4000&2000&3000&2500&4500\\2500&2600&3400&3000&4000\\1200&1800&2600&4100&3000\\2200&2600&3100&3700&3200\end{bmatrix}$$
	- 仓库需求向量  $d=\begin{bmatrix}15&18&14&20\end{bmatrix}$
	- 工厂生产力向量   $s=\begin{bmatrix}20&22&17&19&18\end{bmatrix}$
	- 工厂固定成本  $f=\begin{bmatrix}12000&15000&17000&13000&16000\end{bmatrix}$
	- 编程求解
```python
import gurobipy as gp
from gurobipy import GRB

# 参数定义
transport_cost = [
    [4000, 2000, 3000, 2500, 4500],
    [2500, 2600, 3400, 3000, 4000],
    [1200, 1800, 2600, 4100, 3000],
    [2200, 2600, 3100, 3700, 3200]
]
demand = [15, 18, 14, 20]
supply = [20, 22, 17, 19, 18]
fixed_cost = [12000, 15000, 17000, 13000, 16000]

num_warehouses = len(demand)
num_factories = len(supply)

# 创建模型
model = gp.Model("distribution")

# 创建变量
x = model.addVars(num_factories, num_warehouses, vtype=GRB.CONTINUOUS, name="x")
y = model.addVars(num_factories, vtype=GRB.BINARY, name="y")

# 目标函数
model.setObjective(gp.quicksum(transport_cost[j][i] * x[i, j] for i in range(num_factories) for j in range(num_warehouses)) +
                   gp.quicksum(fixed_cost[i] * y[i] for i in range(num_factories)), GRB.MINIMIZE)

# 约束条件
# 每个仓库的需求必须得到满足
for j in range(num_warehouses):
    model.addConstr(gp.quicksum(x[i, j] for i in range(num_factories)) == demand[j], name=f"demand_{j}")

# 每个工厂的供应不能超过其生产能力
for i in range(num_factories):
    model.addConstr(gp.quicksum(x[i, j] for j in range(num_warehouses)) <= supply[i] * y[i], name=f"supply_{i}")

# 求解
model.optimize()

# 输出结果
if model.status == GRB.OPTIMAL:
    print("\nOptimal solution found:\n")
    for i in range(num_factories):
        if y[i].x > 0.5:
            print(f"Factory {i+1} is open.")
            for j in range(num_warehouses):
                if x[i, j].x > 0:
                    print(f"  Ship {x[i, j].x} units to Warehouse {j+1}.")
        else:
            print(f"Factory {i+1} is closed.")
    print(f"\nTotal Cost: {model.objVal}")
else:
    print("No optimal solution found.")

```
