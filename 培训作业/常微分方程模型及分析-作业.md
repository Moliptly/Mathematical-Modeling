#### 作业要求
在第一次世界大战中，甲、乙两方战役的胜负，往往由双方兵力多少及战斗力强弱两个因素决定，其中，兵力因战斗及非战斗减员而减少，因增援而增加；战斗力与射击次数及命中率有关。请按照正规战争和游击战争两类情况， 应用常微分方程进行建模，分析并预测战役结局。

#### 作业分析
- **正规战模型**：双方兵力的变化主要由战斗减员、非战斗减员和增援决定，可以使用类似于Lanchester 方程的模型
	- Lanchester's Linear Law
		- 甲方兵力：$A(t)$
		- 乙方兵力：$B(t)$
		- 甲方的战斗力系数：$\alpha$
		- 乙方的战斗力系数：$\beta$
	- 方程如下
		- $\frac{dA}{dt}=-\beta B(t)$
		- $\frac{dB}{dt}=-\alpha A(t)$
		- 该模型假设每一方的损失率与敌方的兵力成正比
	- 增加非战斗减员和增援
		- 甲方非战斗减员率：$\delta_A$​
		- 乙方非战斗减员率：$\delta_B$​
		- 甲方增援率：$\gamma_A$​
		- 乙方增援率：$\gamma_B$
	- 修正方程如下
		- $\frac{dA}{dt}=-\beta B(t)-\delta_AA(t)+\gamma_A$
		- $\frac{dB}{dt}=-\alpha A(t)-\delta_BB(t)+\gamma_B$
- **游击战模型**：小规模、高机动性的战斗，游击队的战斗力和损失率与地形、战术等因素有很大关系，可以使用修正后的 Lanchester 方程来描述
	- 修正模型：假设游击队（甲方）的战斗力与其自身兵力成正比，但损失率与敌方兵力和战术系数有关
		- 甲方兵力：$A(t)$
		- 乙方兵力：$B(t)$
		- 甲方的战术系数：$\eta$
		- 乙方的战斗力系数：$\theta$
	- 方程如下
		- $\frac{dA}{dt}=-\theta B(t)-\eta A(t)$
		- $\frac{dB}{dt}=-\alpha A(t)-\delta_BB(t)+\gamma_B$

#### 求解过程
- 设置随机数值模拟战斗场面，通过python编程求解
```python
import numpy as np
from scipy.integrate import odeint
import matplotlib.pyplot as plt

# 参数设置
alpha = 0.01  # 乙方战斗力系数
beta = 0.02   # 甲方战斗力系数
delta_A = 0.001  # 甲方非战斗减员率
delta_B = 0.0015  # 乙方非战斗减员率
gamma_A = 5    # 甲方增援率
gamma_B = 3    # 乙方增援率

# 初始兵力
A0 = 1000
B0 = 1200

# 正规战模型的微分方程
def regular_war(y, t, alpha, beta, delta_A, delta_B, gamma_A, gamma_B):
    A, B = y
    dA_dt = -beta * B - delta_A * A + gamma_A
    dB_dt = -alpha * A - delta_B * B + gamma_B
    return [dA_dt, dB_dt]

# 时间点
t = np.linspace(0, 200, 200)

# 求解ODE
y0 = [A0, B0]
sol = odeint(regular_war, y0, t, args=(alpha, beta, delta_A, delta_B, gamma_A, gamma_B))

# 绘图
plt.plot(t, sol[:, 0], label='甲方兵力 A(t)')
plt.plot(t, sol[:, 1], label='乙方兵力 B(t)')
plt.xlabel('时间 t')
plt.ylabel('兵力')
plt.legend()
plt.title('正规战兵力变化')
plt.show()
```
