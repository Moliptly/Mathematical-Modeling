- **题目二**![[1721449859009.png]]
	- **数据分析**
		- 统计数据中各物料出现的总需求量，并按降序排序
		```python
		import pandas as pd
		
		# 读取Excel文件
		file_path = 'data.xlsx'
		df = pd.read_excel(file_path)
		
		# 计算每个物料编码的总需求量和出现频数
		summary = df.groupby('物料编码').agg({'需求量': 'sum', '物料编码': 'count'})
		summary.rename(columns={'需求量': '总需求量', '物料编码': '频数'}, inplace=True)
		
		# 按照总需求量降序排序
		sorted_summary = summary.sort_values(by='总需求量', ascending=False)
		
		# 显示结果
		print(sorted_summary)
		# 将结果保存到Excel文件
		sorted_summary.to_excel('result.xlsx')
		```
		- 统计结果
		- ![[1721451400472.png]]
		- 选择需求量最大、频数最大的前6个物料
	- **数据处理**
		- 选择频数最大的物料进行分析
		- 将时间转化为**以周数为单位**进行
		- ![[1721530801796.png]]
		-  转化程序如下
		```python
		# 找到第一行的日期并设为第一周 
		start_date = df['日期'].min() df['周数'] = ((df['日期'] - start_date).dt.days // 7) + 1 
		# 按周数和物料编码分组，并计算每组的需求量总和 
		weekly_demand = df.groupby(['周数', '物料编码'])['需求量'].sum().reset_index() 
		# 按周数分组，计算每周的总需求量 
		total_weekly_demand = weekly_demand.groupby('周数')['需求量'].sum().reset_index()
		```
		- 做出物料需求量随时间周数变化图如下![[1721530645054.png]]
		- **使用ARIMA（5，1，0）模型**拟合
			- 训练结果如下![[1721543605464.png]]
			- 可以看到高p值为0.95表示残差序列无显著相关性，模型假设得到支持
			- 大部分回归系数显著，但L5不显著
			- 训练代码如下
			```python
			# 拟合ARIMA模型
			model = ARIMA(weekly_demand, order=(5, 1, 0))  # 可以调整参数 (p, d, q)
			model_fit = model.fit()
			
			# 打印模型摘要
			print(model_fit.summary())
			  
			# 预测未来4周的需求量
			forecast = model_fit.forecast(steps=4)
			print(forecast)
			```
