#### 作业分析
- 状态转移矩阵$A:$
$$A=\begin{bmatrix}0.5&0.2&0.3\\0.3&0.5&0.2\\0.2&0.3&0.5\end{bmatrix}$$
- 观测概率矩阵$B:$
$$B=\begin{bmatrix}0.5&0.5\\0.4&0.6\\0.7&0.3\end{bmatrix}$$
- 初始状态概率向量$\pi:$
$$\pi=\begin{pmatrix}0.2\\0.4\\0.4\end{pmatrix}$$
-  观测序列 $O=($红，白，红，白),设红球对应$O_1=1$ ,白球对应$O_2=2$, 观测序列转换为 $O=(1,2,1,2)$

- 前向算法步骤

#### 实现过程
- 初 始 化 : 
$$\alpha_1(i)=\pi_ib_i(O_1)\quad\text{对于 }i=1,2,3$$
- 递 归 : 
$$\alpha_t(j)=\left[\sum_{i=1}^N\alpha_{t-1}(i)a_{ij}\right]b_j(O_t)$$$t=2,3,\ldots,T$
- 终止:
$$P(O|\lambda)=\sum_{i=1}^N\alpha_T(i)$$

- 通过python编程求解
```python
import numpy as np

# 给定的隐马尔可夫模型参数
A = np.array([
    [0.5, 0.2, 0.3],
    [0.3, 0.5, 0.2],
    [0.2, 0.3, 0.5]
])

B = np.array([
    [0.5, 0.5],
    [0.4, 0.6],
    [0.7, 0.3]
])

pi = np.array([0.2, 0.4, 0.4])

# 观测序列
O = [0, 1, 0, 1]  # 红, 白, 红, 白 (假设红对应0，白对应1)

# 前向算法
def forward_algorithm(A, B, pi, O):
    N = A.shape[0]  # 状态数量
    T = len(O)  # 观测序列长度
    
    # 初始化
    alpha = np.zeros((T, N))
    alpha[0, :] = pi * B[:, O[0]]
    
    # 输出初始化后的alpha
    print(f"Alpha at time 0: {alpha[0, :]}")
    
    # 递归
    for t in range(1, T):
        for j in range(N):
            alpha[t, j] = np.sum(alpha[t-1, :] * A[:, j]) * B[j, O[t]]
        # 输出每次递归后的alpha
        print(f"Alpha at time {t}: {alpha[t, :]}")
    
    # 终止
    P_O_lambda = np.sum(alpha[T-1, :])
    return P_O_lambda, alpha

# 计算观测序列在给定模型下的概率
P_O_lambda, alpha = forward_algorithm(A, B, pi, O)
print("P(O|lambda) =", P_O_lambda)

# 输出整个alpha矩阵
print("\nAlpha matrix:")
print(alpha)
```
