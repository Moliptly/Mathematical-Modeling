#### 作业分析
- **题目一**
	- **问题陈述**：需要确定从直径为1的圆木中切割出的矩形梁的尺寸（长 l 和宽 w），以使重量最轻且强度最大
	- **数学建模**
		- 几何约束：$l^2+w^2\leq1$
		- 目标
			- 最小化重量：重量与截面积$A = l \times w$ 成正比
			- 最大化强度：强度与惯性矩$S=\frac{wl^3}{12}$相关
	- **多目标优化问题**$$\begin{aligned}&\min_{l,w}-\frac{wl^{2}}{12} \\&\operatorname*{min}_{l,w}lw \\&\text{约束条件:} \\&l^{2}+w^{2}\leq1 \\&l,w\geq0\end{aligned}$$
	- **编程求解**
	```python
	import numpy as np
	from scipy.optimize import minimize
	import matplotlib.pyplot as plt
	# 定义目标函数
	def objective(x, w1, w2):
	    a, b = x
	    A = a * b
	    I = (a * b**3) / 12
	    return w1 * A - w2 * I  # 组合目标函数
	
	# 定义约束条件
	def constraint(x):
	    a, b = x
	    return a**2 + b**2 - 1  # 几何约束
	    
	# 初始猜测值
	x0 = [0.5, 0.5]
	
	# 定义约束字典
	cons = ({'type': 'eq', 'fun': constraint})
	
	# 定义权重
	weights = np.linspace(0, 1, 100)
	results = []
	optimal_result = None
	optimal_value = float('inf')
	
	for w1 in weights:
	    w2 = 1 - w1
	    result = minimize(objective, x0, args=(w1, w2), constraints=cons, bounds=((0, 1), (0, 1)))
	    a_opt, b_opt = result.x
	    A = a_opt * b_opt
	    I = a_opt * b_opt**3 / 12
	    results.append((A, I))
	
	    # 记录最优结果
	    current_value = objective(result.x, w1, w2)
	    if current_value < optimal_value:
	        optimal_value = current_value
	        optimal_result = (a_opt, b_opt, A, I)
	
	# 提取结果
	areas, inertias = zip(*results)
	
	# 绘制Pareto前沿
	plt.figure(figsize=(12, 6))
	plt.plot(areas, inertias, 'o-')
	plt.xlabel('Cross-sectional area')
	plt.ylabel('Moment of inertia')
	plt.title('Pareto Frontier')
	plt.show()
	
	# 输出最优结果
	a_opt, b_opt, A_opt, I_opt = optimal_result
	print(f'最优截面长: {a_opt}')
	print(f'最优截面宽: {b_opt}')
	print(f'最优截面积: {A_opt}')
	print(f'最优惯性矩: {I_opt}')
	```

- **题目二**
	- **问题陈述**：需要确定在4年内如何使用400万元资金，以使总效益最大化，考虑效益函数 $\sqrt{x}$​ 和年利率为10%
	- **数学建模**
		- **变量**：$x_i\text{:第 }i\text{ 年使用的资金 }(i=1,2,3,4)$
		- **目标**：$\text{最大化4年的总效益:}\sum_{i=1}^4\sqrt{x_i}$
		- **约束条件**
			- 4年内使用的总资金必须为400万元
			- 每年的资金可以以10%的利率结转
			- 每年使用资金x万元，则可得效益是 $\sqrt{x}$ 万元
		- **递归资金约束**
			- $x_{i}\leq F_{i-1}$
			- $F_{i}=(F_{i-1}-x_{i})\times1.1$对于$i=2,3,4$
			- $F_0=4000000$
		- **优化问题**为$$\begin{aligned}&\max_{x_1,x_2,x_3,x_4}\sum_{i=1}^4\sqrt{x_i} \\&\text{约束条件} \\&F_0=4000000 \\&F_i=(F_{i-1}-x_i)\times1.1\quad\text{对于 }i=2,3,4 \\&F_{4}=0 \\&x_i\leq F_{i-1}\quad\text{对于 }i=1,2,3,4 \\&x_i\geq0\quad\text{对于 }i=1,2,3,4\end{aligned}$$
		- 动态规划公式：$f(n,k)=\max_{0\leq x\leq k}(\sqrt{x}+f(n+1,(k-x)\times1.1))$
		- 其中：
			- $f(n, k)$表示在第 n 年，剩余资金为 k 万元时的最大效益
			- 初始条件：$f(4, k) = 0$（第4年之后没有效益）
		- **编程求解**
	```python
	import numpy as np
	# 定义年份和资金量
	years = 4
	total_funds = 400
	interest_rate = 0.1
	
	# 初始化动态规划表格
	f = np.zeros((years + 1, total_funds + 1))
	
	# 动态规划求解
	for year in range(years - 1, -1, -1):
	    for fund in range(total_funds + 1):
	        max_profit = 0
	        for x in range(fund + 1):
	            remaining_fund = fund - x
	            next_fund = min(int(remaining_fund * (1 + interest_rate)), total_funds)  # 确保索引不超过数组范围
	            current_profit = np.sqrt(x) + f[year + 1, next_fund]
	            if current_profit > max_profit:
	                max_profit = current_profit
	        f[year, fund] = max_profit
	
	# 输出最优效益
	optimal_profit = f[0, total_funds]
	print(f"最大效益: {optimal_profit:.2f} 万元")
	
	# 追踪资金使用计划
	fund = total_funds
	usage_plan = []
	  
	for year in range(years):
	    max_profit = 0
	    best_x = 0
	    for x in range(fund + 1):
	        remaining_fund = fund - x
	        next_fund = min(int(remaining_fund * (1 + interest_rate)), total_funds)  # 确保索引不超过数组范围
	        current_profit = np.sqrt(x) + f[year + 1, next_fund]
	        if current_profit > max_profit:
	            max_profit = current_profit
	            best_x = x
	    usage_plan.append(best_x)
	    fund = int((fund - best_x) * (1 + interest_rate))
	
	# 输出资金使用计划
	for year, usage in enumerate(usage_plan, start=1):
	    print(f"第 {year} 年使用资金: {usage} 万元")
	```
