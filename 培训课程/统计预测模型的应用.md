#### 数据预测问题
- 根据输出变量的类型
	- **分类预测**：输出变量为分类型、离散型
	- **回归预测**：输出变量为数值型、连续型
	- **时间序列预测**：按照时间序取得一系列观测值，相邻观测值之间具有依赖性
- **注意**
	- 预测需要数据
	- 预测随机事物：从随机变化规律中，归纳出相对固定的模式
	- 短期可预测：时间太长可行度较低
	- 预测结果不确定性
- 变量
	- 输入变量：特征变量，自变量，x，通常多个
	- 输出变量：因变量，y，通常一个
- 建模过程
	- 基于数据集构造特征集
	- 根据预测目标选择预测方法
	- 建立预测模型并优化
	- 一般过程
		- 确定问题
		- 数据准备
		- 分析规律（趋势性、周期性、波动性、相关性、相似性、关联性）
		- 建立预测模型
			- 特征构建、特征选择、算法选择
			- 构建模型、测试模型、模型优化
		- 评估模型
			- 评估回归模型（均方差、均方根误差）
			- 评估分类模型（误差率、错误率、混淆矩阵）
		- 模型比较（应用不同模型并选择最终的模型）

#### 一元线性回归
- 例子
	- $$y=1.5x_1-3.2+\epsilon\ \ \ \ \ \epsilon\sim N(0,1.2^2)$$
	- $\epsilon$ 为噪声，服从正态分布
	- 结果解读
		- p值越小，显著性越高，线性特征越明显
	- 残差分析
		- 残差：实际值与预测值之间的差异
		- 线性回归模型假设残差具有同方差性，正态分布
		- 检验模型残差
			- 点预测：计算均方误差
			- 区间预测：置信区间


#### 回归预测
- 二手汽车价格问题举例
	- 输出变量：价格
	- 输入变量：17个属性（不列举了）
	- 分析
		- 定性输入与定量输入变量之间的关系
		- 定量输入与定量输入变量之间的关系
		- 计算变量之间的相关系数(热力图)
			- 变量之间的相关性越低，线性特征越好
			- 删除两个相关性较高的其中一个变量
			- 删除由其他特征线性组合的特征
	- 划分：将85%划分为训练集，将15%划分为测试集
	- 特征选择
		- 向后选择：从完整模型开始，逐个去除特征，最大改善模型评分
	- 问题
		- 去掉更多特征之后，预测均方差未得到改善，如何优化？
			- 正则化：减少变量数防止过拟合
				- 岭回归
				- Lasso回归：最小绝对值收缩和选择算子
